The goal of this code is to create a rock, paper, scissors game, located and played in the console only.

SET-UP THE GAME 
Define User Name
Welcome user
Create a scoreboard
Determine User Choice
Determine Computer Choice

RUN THE GAME 
Compare choices
Announce the winner
Increment the score
Increment the round

SHARE THE RESULTS
Announce Winner
Show scoreboard
Announce next round number
Start next round or End the set

userName
promptUserSelection
userSelection 
computerSelection
userScore
computerScore
gameRound

LIST OF FUNCTIONS:
