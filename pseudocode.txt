The goal of this code is to create a rock, paper, scissors game, located and played in the console only.

SET-UP THE GAME 
Define User Name
Post rules
Create a scoreboard
Determine User Choice
Determine Computer Choice

RUN THE GAME 
Compare choices
Announce the winner
Increment the score
Increment the round

SHARE THE RESULTS
Announce Winner
Show scoreboard
Announce next round number
Start next round or End the set

LIST OF GLOBAL VARIABLES:
userName
promptUserSelect
userSelection 
computerSelect
userScore
computerScore
gameRound

LIST OF FUNCTIONS:
